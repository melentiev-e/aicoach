# Phase review

The reassessment skill — also the "update the program to current shape" skill. Two
triggers, same logic: scheduled every 4–6 weeks, or on demand when fitness moves
materially. Don't chase a single workout; act on sustained movement in the markers.

## Read first
- `CLAUDE.md` §5 (methodology + zones), §8 (adjust paces / adjust goal).
- `logs/fitness.csv` (the dated trajectory — read the trend, not just the last row).
- `plan/master-plan.md`, recent `reviews/`.
- `athlete.md` and §3 fitness snapshot block.

## Steps

1. **Re-read the trajectory.** From `fitness.csv`: VO2max trend, Marathon Shape, race
   prognosis, CTL/ATL/TSB. Where is the athlete relative to the phase plan?

2. **Re-derive pace zones** (§5) from the new fitness number if it moved materially.
   Update the zone table in `CLAUDE.md` and the §3 snapshot. Re-derive — don't nudge.

3. **Revise the master plan.** Adjust `plan/master-plan.md` for the next block: phase
   focus, volume band, key workout types. Treat it as a living document — revise and
   date the change, don't rewrite from scratch.

4. **Re-judge the goal (§8).** If across multiple checkpoints the prognosis and key
   workouts consistently contradict 2:39, say so plainly and propose a revised target or
   extended timeline. If on track, confirm it with the evidence.

5. **Consider a tune-up race** if the phase calls for one.

6. Note the reassessment in `reviews/YYYY-MM-DD-phase.md`.

## May write
`plan/master-plan.md`, pace zones + §3 snapshot in `CLAUDE.md`, `athlete.md`,
`reviews/YYYY-MM-DD-phase.md`.
